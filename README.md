# KL Dashboard

A Dashboard project to control Emmet clients.




## Features

- FTP Server
- VNC Viewer
- Console
- Client Updater
- AutoRefresh


## Deployment

**Required modules:**
- express
- express-session
- express-fileupload
- ejs
- socket.io
- fs

**To install dependencies run:**

```bash
  npm -i
```

**To start dashboard run:**

```bash
  npm start
```
or
```bash
  node index.js
```
